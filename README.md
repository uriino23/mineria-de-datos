# Índice de Prácticas y Ejercicios

Este repositorio contiene los scripts de código, bases de datos y archivos de apoyo organizados de manera consecutiva. Haz clic en cualquiera de los enlaces para acceder directamente al archivo.

* [Ejercicio 1: Operación aritmética básica](<%23 Calcular la operación siguiendo el ord.py>) - Script para calcular operaciones siguiendo el orden de precedencia.
* [Ejercicio 2: Lectura e impresión de registros CSV](abrir_archivoscsv.py) - Script para la lectura y visualización de archivos separados por comas.
* [Ejercicio 3: Análisis estadístico de calificaciones](actividad_calificaciones.py) - Script para el análisis de rendimiento escolar.
* [Ejercicio 4: Cálculo de área y volumen](area_y_volumen.py) - Script para el cálculo de áreas y volúmenes geométricos.
* [Ejercicio 5: División con cociente y residuo](calculadora.py) - Script de calculadora enfocado en operaciones de división.
* [Ejercicio 6: Manejo de listas de datos externos](calificaciones%20_lista.py) - Script enfocado en la manipulación y uso de listas de datos.
* [Ejercicio 7: Base de datos escolar (CSV)](calificaciones.csv) - Archivo de datos con registros de calificaciones estudiantiles.
* [Ejercicio 8: Bucle repetidor de palabras](ciclo.py) - Script interactivo basado en bucles repetitivos.
* [Ejercicio 9: Validación de contraseña secreta](contraseña.py) - Script para la verificación y validación de contraseñas.
* [Ejercicio 10: Lector de ventas con diccionarios](diccionario.py) - Script estructurado con diccionarios para el control de ventas.
* [Ejercicio 11: Operaciones de división](división.py) - Script enfocado en el manejo de operaciones de división.
* [Ejercicio 12: Validación de edad](edad.py) - Script de condicionales para verificar la edad de un usuario.
* [Ejercicio 13: Función para cálculo de IVA](iva.py) - Script modular con funciones para calcular el impuesto al valor agregado.
* [Ejercicio 14: Pizzería Bella Napoli / Pedidos](pizza.py) - Script de simulación para la toma y gestión de pedidos.
* [Ejercicio 15: Verificación de número primo](primo.py) - Script enfocado en algoritmos numéricos para identificar números primos.
* [Ejercicio 16: Script interactivo de saludo](saludo.py) - Script básico de interacción y entrada/salida de datos.
* [Ejercicio 17: Registro de datos comerciales (CSV)](ventas.csv) - Archivo de datos con el historial de ventas comerciales.
* [Bloque 2 - Tendencia Central 1](tendecia_central_1.py) - Script de código sobre medidas de tendencia central.
* [Bloque 2 - Medidas de Dispersión 2](medidas_de_dispersion_2.py) - Script de código para el cálculo de dispersión.
* [Bloque 2 - Correlación 3](correlacion_3.py) - Script enfocado en análisis de correlación.
* [Bloque 2 - Tablas 4](tablas_4.py) - Script para la generación y manejo de tablas de datos.
* [Bloque 2 - Árbol 5](arbol_5.jpg) - Archivo gráfico (Imagen JPEG) relacionado con estructuras de árbol o diagramas.
* [Bloque 2 - Ejercicio 6](ejercicio_6.jpg) - Enunciado o resolución gráfica (Imagen JPEG) del ejercicio 6 del bloque 2.
* [Bloque 2 - Ejercicio 7](ejercicio_7.py) - Script de código correspondiente al ejercicio 7 del bloque 2.
